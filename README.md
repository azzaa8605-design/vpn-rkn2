# VPN-RKN Android (Phone-friendly build)

This repo contains an Android app skeleton named **VPN-RKN**.

## Get an APK from your phone (no Android Studio)
1. Create a new GitHub repo.
2. Upload this repository contents.
3. Open **Actions** → run **Build APK**.
4. Download artifact **VPN-RKN-APK** → install the `.apk` on your phone.

Notes:
- The APK is built on GitHub runners using Android SDK.
- This project currently is a UI skeleton (Connect/Disconnect button). VPN core/TUN logic is not implemented yet.
